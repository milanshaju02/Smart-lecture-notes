# Smart Lecture Notes

AI-powered lecture note generation system that converts lecture audio into structured and meaningful academic notes.

## 🚀 Features
- 🎤 Speech-to-text conversion using Whisper  
- 🧠 NLP-based lecture summarization  
- 📌 Generation of structured important points  
- 🌐 Multi-language support  
- 📄 Automatic PDF note generation  
- ⏱ Timestamp-based lecture segmentation  
- ❓ MCQ and flashcard generation  

---

## 🛠 Tech Stack
- Python (Flask)  
- Faster-Whisper (Speech Recognition)  
- Transformers (BART, FLAN-T5)  
- ReportLab (PDF generation)  
- Deep Translator
